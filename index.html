How could increased adoption of open data standards help cities? This project explored several cases to try to understand and demonstrate the value — beyond common examples like GTFS. Read more in our explanatory blog post [add link].

Created by Andrew Yaspan, a 2017 Open Cities Fellow at the Sunlight Foundation.



Mapping data across municipalities in a metro area

Mapping points from datasets containing the same type of information from multiple cities within a region.


Providing the ability to have “plug-and-play” tools

Creating a map and some basic graph visualizations for any dataset following a particular data standard to discover trends particular to that city.


Making city-to-city comparisons

Designing a visualization made for the express purpose of viewing the data of cities in a context that provides insight on trends based on similarities or differences between the cities.
