--- layout: default title: "Providing the ability to have “plug-and-play” tools" ---

One way cities could benefit from data standards is by gaining access to tools that use that data for some meaningful application — "plug-and-play" tools that work for any city whose data is in a particular format. Right now, creating online visualizations and applications that use open data is a big effort for most cities, one that there is often little capacity for. If data portals came with a set of built-in features that make open data easier to understand — that can be used by any place that uses certain data standards — cities could make their data more user-friendly and useful quickly, easily, and affordably.

As an example, I created a tool that generates a map and visualizations of building permits, as long as the data conforms to the BLDS standard, with examples below for San Diego County (unincorporated areas) and Chattanooga.
